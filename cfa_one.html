Chick-Fil-A One Rewards Calculator

Category : Java

Chick-fil-A One™ is a tiered membership program that gives users the ability to earn points with every purchase made at Chick-fil-A through their smartphone. This is a Java problem proposal/solution that will allow users to input point values and receive more information and redeemable menu items computed by the program.
